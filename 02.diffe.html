Difference between Relative and Absolute positioning

Relative position-

The element is positioned Relative to its normal position.When you add an additional position apart from relative ,such as left:20px the element will move 20px to the right from its normal position. syntax.. position: relative;

Absolute Position-

The element is positioned absolutely to its first postioned parent.in the case when there is no positioned parent element, the element that has position set to absolte will be postioned related directly to the html element. syntax.. position: absolute;
